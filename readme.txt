== Description ==

Use case is for creating supervised working groups for pairs of students.
These should be 2 or more students supervised by teachers.

In this use case:
 - the site is intended to be private members site only, no public groups are allowed
 - Pair groups are unlisted Hidden groups which have student members and supervisor teachers
 - Teachers are allowed to act on behalf of students within the same school
